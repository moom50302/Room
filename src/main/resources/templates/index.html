    📋 討論區 — 開發歷程

    🏗️ Day 1 — 核心功能建立

    討論基礎

    • 暱稱輸入 → 大廳 → 討論區 三層畫面切換
    • WebSocket (STOMP over SockJS) 即時訊息收發
    • 斷線通知、線上使用者 IP + 暱稱列表

    多討論區大廳系統

    • Ticket 討論區 — 輸入 Redmine 單號，自動組成 URL
    • Normal 討論區 — 自訂名稱
    • 重複單號檢查、建立後自動加入
    • 空討論區自動刪除（含所有關聯資料）

    討論區內功能 Tabs

    • 💬 討論 — Markdown 渲染、@提及使用者、表情符號選擇器、圖片上傳貼圖
    • 🗳️ 投票 — 發起投票、單次投票限制、即時票數長條圖
    • 📅 行事曆 — 新增/刪除事件、依日期排序
    • 🎰 樂透選 — 輸入選項隨機抽選
    • 🎨 畫布 — 多人共同塗鴉、顏色/粗細選擇

    資料同步機制

    • RoomHistoryService 累積討論紀錄、樂透結果、畫布筆跡
    • 新加入者自動收到所有歷史資料
    • 投票/行事曆/樂透更新同步顯示系統訊息到討論區
    • Tab 未讀徽章計數，切換後清除

    📌 Day 2 — 互動與增強功能

    使用者互動系統

    • 點擊名單彈出互動選單：👋 打招呼、🤝 握手、❤️ 愛心、😠 抗拒、👉 戳
    • 互動訊息發言者統一為「互動」
    • 🎮 猜拳 — 剪刀石頭布挑戰，等待對手回應，結果公告到討論區，名單旁顯示戰績

    討論區增強

    • Markdown 語法支援（粗體、斜體、刪除線、程式碼、超連結）
    • 表情符號選擇器 — 90 個常用 emoji
    • 圖片上傳貼圖功能
    • ℹ Tips 提示面板 — 顯示 @tag 與 Markdown 語法說明
    • Ticket 討論區專屬：下載討論紀錄為 .txt

    畫布改進

    • 🧹 橡皮擦 — 可調大小，動態圓形外框游標顯示擦除範圍
    • 畫筆座標修正（CSS/Canvas 像素 scale factor）

    邀請功能

    • 輸入多筆 Email，產生含直連 URL 的邀請文字供複製
    • Email 在 URL 中以 Base64 編碼，避免明文暴露
    • 收到連結者開啟後自動帶入暱稱、直接加入討論區

    📌 Day 3 — 新 Tab 與資料管理

    🔮 運勢（新 Tab）

    • 六十甲子籤完整收錄（來源：北港朝天宮）
    • 籤筒搖晃動畫 → 金色籤滑出 → 運勢卡淡入
    • 顯示籤號天干地支、運勢等級（彩色標籤）、四句籤詩、白話解籤
    • 所有人同步看到相同結果，動畫結束後才發送討論通知

    🎰 樂透選改進

    • 新增轉盤旋轉動畫，所有人同步觀看
    • 討論通知延遲到動畫結束後才顯示

    🎨 畫布快照系統

    • 儲存畫布 — 保存當前畫面，縮圖排列在畫布下方
    • 點擊縮圖 — 放大預覽 modal
    • 下載成 PNG — 匯出為圖片檔
    • 重現畫布 — 清除當前內容，載入選取的快照

    📝 Ticket 討論區 Memo 功能

    • 討論輸入旁新增 Memo 按鈕（僅 Ticket 討論區顯示）
    • 文字渲染成圖片卡片，排列在下方
    • 點擊放大檢視，可刪除
    • 討論區僅顯示系統訊息通知

    離開討論區確認

    • 最後一人離開時跳出確認提醒：所有紀錄將被刪除
    • 進入新討論區時清空畫布快照、運勢狀態等前端資料

    🎨 UI 主題化

    全站紫色漸層主題

    • 背景採用淡紫色漸層 (#f5f3ff → #ede9fe)
    • 所有容器改為白色圓角卡片 + 陰影，無邊框
    • 標題文字使用紫色漸層 (#667eea → #764ba2)
    • 按鈕統一紫色漸層，hover 加深 + 陰影
    • Tab、輸入框、邊框統一紫色調
    • 輸入框 focus 狀態加紫色光暈
    • 開發歷程按鈕固定右下角

    大廳 UX 優化

    • 建立流程整合為「建立討論區」按鈕 + Modal 彈窗
    • Modal 內可切換 Ticket 討論區 / Normal 討論區
    • 主畫面輸入格改為討論區搜尋（輸入 ≥ 2 字元自動篩選）
    • 建立 Tab 未選中顏色調亮，hover 回饋優化

    📅 Day 4 — 一番賞抽獎討論區

    🎰 一番賞抽獎討論區（新房型）

    • 新增第三種討論區類型「一番賞抽獎 討論區」
    • 一番賞房間只顯示「討論」和「抽賞」兩個 Tab
    • 建立抽賞：設定名稱、多個獎賞（名稱+數量）、每 IP 抽獎上限
    • 抽賞列表：顯示所有抽賞卡片，含剩餘獎品統計
    • 抽賞詳情：獎賞表格含剩餘數量，抽獎 + 刪除按鈕
    • 摸彩動畫：彩球跳動 → 中獎球彈出 → 顯示結果
    • 結果同步到討論區（動畫結束後才顯示）
    • IP 抽獎次數限制 + 獎品抽完提示
    • 只有建立者可以刪除抽賞
    • 新加入者自動同步抽賞資料

    🎲 畫布隨機線圖

    • 畫布工具列新增「隨機線圖」按鈕
    • 內建 20 種線稿（貓、花、星星、房子、魚、蝴蝶、樹、愛心、太陽、月亮、雲朵、雨傘、鑽石、音符、杯子、船、鈴鐺、蘋果、閃電等）
    • 所有人同步可見，可在線稿上繼續塗鴉

    🔔 新訊息通知

    • 討論區有新訊息時，頁面標題閃爍「💬 (N) 新訊息！」
    • 搭配音效提示（Web Audio API 合成，無需音效檔案）
    • 僅在分頁不在前景時觸發（Page Visibility API）
    • 不通知自己發的訊息和系統訊息（JOIN/LEAVE）
    • 累計未讀數顯示在標題中，回到分頁自動恢復原標題
    • 100% 相容任何環境，不依賴 HTTPS

    🎨 主題切換系統

    • CSS 全面改用 Custom Properties（CSS 變數）
    • 支援主題切換：💜 寧靜 / 🔥 火焰 / 🌿 青草 / 🌊 浪花 / 🌸 櫻花綻放
    • 火焰主題：深色背景、橙紅漸層、暗色卡片、流動火焰邊框、火星粒子
    • 青草主題：藍天白雲頂部、青草大地底部、飄動雲朵動畫、微風吹拂草地動畫、翡翠綠主色
    • 浪花主題：淡藍海洋漸變淺黃沙灘、雙層波浪拍打動畫、白色浪花泡沫、淡藍主色
    • 幾何主題：白色背景、20 個粒子做物理反彈運動、200px 內自動連線、灰藍主色
    • 櫻花綻放：粉色漸層背景、右側 CSS 繪製櫻花樹（樹幹+枝幹+花叢）、35 片花瓣 Canvas 動畫隨風往左飄落（Bezier 曲線花瓣形狀、自旋+擺動）
    • 主題選擇記憶在 localStorage，重整不會重置
    • 切換按鈕位於右下角（開發歷程上方）

    🛠️ 技術架構

    框架Spring Boot 2.7.18 + Java 8
    即時通訊STOMP over SockJS (WebSocket)
    前端模板Thymeleaf
    建構工具Gradle Kotlin DSL
    資料儲存ConcurrentHashMap（記憶體內）
    前端 JS~2000 行
    CSS~2100 行
    後端 Controller10 個
    後端 Service7 個
